Recherche du mot 开放

ÉTAPE 1 : Constitution du corpus

  • 19, Décembre 2024
  • POSTED BY Xiaotong
  • 开放

Le mot ouverture en chinois est “开放”, j'ai choisi d'étudier le mot “开放” ainsi que ses synonymes et expressions associées en chinois. Ce mot peut apparaître dans différents contextes tels que “博物馆开放参观” (ouverture des musées au public), “开放获取代码” (accès libre au code source), “公园的花朵开放” (les fleurs du parc s'épanouissent) ou encore “改革开放” (la réforme et l'ouverture). Ces expressions montrent une grande variété d'usages du mot dans des domaines tels que la culture, la technologie, la nature et la politique. Pour constituer mon corpus, j'ai utilisé des moteurs de recherche comme Google et Bing afin de collecter des textes pertinents contenant le mot “开放” et ses variantes. Pendant le processus d'analyse, j'ai découvert que le terme “开放” est le plus souvent employé dans le contexte de “改革开放”, une réforme politique et économique majeure ayant eu lieu en 1978, qui revêt une importance extraordinaire pour la Chine contemporaine. Ce projet vise à explorer les différents usages de “开放” dans la langue chinoise, en mettant en lumière sa polyvalence et son rôle dans divers champs sémantiques et culturels.

ÉTAPE 2 : Création du tableau

  • 03, JAN 2025
  • POSTED BY Xiaotong
  • 开放

Selon les indications fournies par la professeure dans le cadre du projet groupe, pour créer le tableau des URLs récupérées, j’ai d’abord écrit un code HTML avec les colonnes qui doivent apparaître dedans, c’est-à-dire : le numéro de ligne, les URLs de la page, son code HTTP, son encodage, le nombre d’occurrences de “开放”, le lien vers les contextes autour des mots et le lien vers le concordancier. Je suis ensuite passée à l’écriture du script bash en créant des variables pour le fichier de sortie HTML, pour les chemins de chacun des dossiers, et pour un compteur de lignes. Je récupère la page avec cURL que je stocke dans mon dossier aspirations et la réponse donnée dans une autre variable afin de la réutiliser pour extraire le code HTTP et l’encodage. Pour le code HTTP, j’ai rajouté une condition pour que la page ne soit pas traitée si le code est différent de 200, et une ligne au tableau qui indique dans chaque colonne qu’aucune donnée n’a été extraite. J’utilise ensuite la commande lynx pour stocker le texte brut de la page dans le dossier dumps-text. Je peux ensuite placer mes variables à l’intérieur du code HTML du tableau et incrémenter le compteur dans la boucle pour que tous les fichiers et le tableau soient générés correctement.

ÉTAPE 3 : Création du concordancier

  • 04, JAN 2025
  • POSTED BY Xiaotong
  • 开放

Dans le cadre du projet, j'ai réalisé un traitement des données en utilisant un script Bash pour extraire le mot "开放" et ses variantes dans les contextes textuels. À partir des pages web récupérées via cURL, j'ai utilisé **lynx** pour convertir les contenus HTML en texte brut et **grep** pour isoler les contextes pertinents, avec deux lignes avant et après chaque occurrence du mot. Ces contextes sont stockés dans des fichiers texte dans un dossier dédié. J'ai également généré un concordancier en HTML, listant les occurrences du mot avec leurs contextes gauche et droit, grâce à des commandes **sed** et **awk**. Les résultats sont organisés en tableaux et accessibles dans des fichiers distincts, facilitant ainsi l'analyse des données.

ÉTAPE 4 : Mise en place du site web

  • 11, JAN 2025
  • POSTED BY Xiaotong
  • 开放

Dans cette étape, j’ai travaillé sur l’uniformisation des styles de tableau avec les membres de l’équipe et sur la modification du site web. Tout d’abord, j’ai adapté les tableaux HTML générés selon le modèle fourni par mes coéquipiers afin de garantir une cohérence visuelle à travers tout le projet. Cela inclut l’application d’un thème de couleurs et de polices unifié, ainsi que l’utilisation du même fichier CSS (par exemple Bulma) pour harmoniser l’apparence générale.
