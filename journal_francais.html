Recherche du mot "ouverture"

ÉTAPE 1 : Constitution du corpus

  • 09, JAN 2025
  • POSTED BY CATALIN
  • ouverture

Le mot "ouverture" en français porte de nombreux sens. Son sens le plus général est celui de la notion d'ouverture qui s'oppose à la fermeture. Cependant, "ouverture" a également d'autres significations dédiées à des domaines spécifiques comme la photographie, la philosophie, la politique, etc. Pour constituer mon corpus, j'ai cherché le mot "ouverture" sur internet et j'ai selectionné à la fois des articles de presse mais également des sites spécialisés dans certains des domaines cités précédemment, le but étant d'avoir des occurrences des significations variés du mot "ouverture".

ÉTAPE 2 : Création des tableaux et des fichiers pour l'analyse

  • 10, JAN 2025
  • POSTED BY CATALIN
  • ouverture

L'une des étapes principales du projet était la création d'un tableau au format html contenant des informations sur les différentes pages qui composent le corpus. Afin d'obtenir ce tableau il a fallu attribuer un numéro de ligne à chaque page et récupérer son lien, son code HTTP, son encodage, le nombre de mots qu'elle contient, le nombre d''occurrences du mot "ouverture" (ainsi que ses variations "Ouverture", "Ouvertures" et "ouvertures"), le lien vers les contextes dans lesquelles chacune des occurrences est trouvée et le lien vers le concordancier, le tout avec un script bash. Les principales commandes à utiliser étaient cURL pour récupérer les pages web et lynx pour récupérer le contenu textuel des pages. Il a également fallu récupérer les contextes dans lesquelles "ouverture" apparaissait et les mettre dans des fichiers spécifiques, ainsi que créer un concordancier pour voir les mots qui précèdent et suivent "ouverture" dans les pages web du corpus. Après celà, il a fallu, à l'aide d'un script bash, tokéniser à la fois le texte brut des pages mais aussi le texte des fichiers contextes afin de préparer l'analyse avec les scripts PALS.

ÉTAPE 3 : Analyse avec PALS

  • 11, JAN 2025
  • POSTED BY CATALIN
  • ouverture

Pour analyser les pages web contenant le mot "ouverture" il a fallu utiliser les scripts PALS. A l'aide du script cooccurrents.py il a été possible d'obtenir les cooccurrents du mot "ouverture". J'ai appliqué le script à la fois pour le texte brut mais aussi pour les contextes dans lesquels on retrouve le mot "ouverture" afin de pouvoir comparer.

ÉTAPE 4 : Visualisation avec un nuage de mots

  • 11, JAN 2025
  • POSTED BY CATALIN
  • ouverture

Afin d'obtenir une représentation visuelle globale des pages contenant le mot "ouverture", j'ai utilisé la commande wordcloud_cli afin d'obtenir deux nuages de mots : un basé sur le mot "ouverture" en contexte et l'autre basé sur les pages brutes. Pour cela, il a d'abord fallu éliminer les stopwords des fichiers avec un script python afin de ne garder que les mots pertinents. Ensuite, l'exécution de la commande wordcloud_cli a permis de créer les nuages de mots afin d'avoir une vision générale des mots les plus fréquents dans le cas des pages entières et pour les contextes où on retrouve obligatoirement le mot "ouverture".
